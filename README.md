# Faiq Wajahat - Portfolio

A premium, minimalistic, and dynamic single-page portfolio built with modern web technologies.

## Tech Stack
- **Framework:** Next.js (App Router)
- **Language:** JavaScript (JSX)
- **Styling:** Tailwind CSS V4
- **Animations:** Framer Motion
- **Icons:** Lucide React

## Features
- **Dark Mode Aesthetic:** Custom curated dark theme focusing on deep blacks and a sleek primary blue accent.
- **Smooth Scrolling Navigation:** A glassmorphic, fixed navbar allows seamless jumps to various sections.
- **Animated Components:**
  - **Hero:** Staggered fade-ins and an interactive call-to-action button.
  - **About:** A responsive Bento-box grid detailing the MERN stack.
  - **Projects:** CSS Grid layout with scroll-reveal cards.
  - **Experience:** A vertical sliding timeline.
  - **Contact:** An elegant call-to-action section with subtle glow effects and interactive social links.

## Getting Started

First, install the dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Structure
All UI components are located within `app/components/` and are rendered dynamically inside the main `app/page.js` file. Global styling variables reside in `app/globals.css`.
